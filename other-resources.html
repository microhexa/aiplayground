This tool was primarily inspired by:

Quick, Draw!

Link

A playful Google game where you draw a picture and an AI tries to guess what it is before time runs out. It shows how computers can learn to recognize simple doodles.

DoodleIt

Link

A drawing game that invites children to sketch objects and see how a computer responds. It makes AI feel more hands-on by turning learning into a simple doodling activity.

Teachable Machine

Link

A beginner-friendly tool from Google that lets people train their own image, sound, or pose models with examples. It is useful for showing that machine learning can be taught by giving a computer examples.

Machine Learning for Kids

Link

A platform designed to help children explore machine learning through easy projects and creative experiments. It connects AI ideas with playful making, coding, and testing.
